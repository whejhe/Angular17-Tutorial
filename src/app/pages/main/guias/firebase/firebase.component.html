Introducción

Firebase es una plataforma de desarrollo de aplicaciones que ofrece varios servicios como autenticación, base de datos en tiempo real, Firestore, almacenamiento en la nube, y hosting. Integrar Firebase en un proyecto de Angular 18 permite aprovechar estas funcionalidades de manera sencilla y eficiente.

Requisitos Previos

  • Node.js y npm instalados.
  • Angular CLI instalado ({{installAngular}}).
  • Cuenta en Firebase y acceso a la consola de Firebase.

Configuración del Proyecto Firebase

  1. Crear un Proyecto en Firebase:

    Ve a la Consola de Firebase. Haz clic en "Add project" y sigue las instrucciones.

  2. Configurar una Aplicación Web en Firebase:

    En la consola de Firebase, selecciona tu proyecto. En el panel izquierdo, selecciona "Authentication" y habilita los métodos de autenticación que necesitas. Selecciona "Firestore Database" y crea una base de datos. Selecciona "Storage" y configura tus reglas de almacenamiento. Selecciona "Hosting" y configura tu dominio.

Instalación y Configuración de Firebase en Angular 18

  1. Crear un Proyecto Angular:

    ng new mi-proyecto-angular
cd mi-proyecto-angular

  2. Instalar Firebase y AngularFire:

    {{installfirebase}}

  3. Configurar Firebase en Angular:

    En la consola de Firebase, ve a "Configuración del proyecto" y copia la configuración de Firebase. Crea un archivo src/environments/environment.ts con el siguiente contenido:

    {{envirotment}}

  4. Inicializar Firebase en el Módulo Principal:

    {{iniciarFirebase}}

Deploy a Firebase Hosting

  1. Instalar Firebase CLI:

    npm install -g firebase-tools

  2. Iniciar Sesión en Firebase:

    firebase login

  3. Inicializar Firebase en tu Proyecto:

    firebase init

    Selecciona "Hosting" y sigue las instrucciones.

  4. Construir el Proyecto Angular:

    ng build --prod

  5. Deploy a Firebase Hosting:

    firebase deploy
